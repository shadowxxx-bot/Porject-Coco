# Coco Match

The easiest way to find your startup co-founder.

## Setup

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

The app runs at `http://localhost:5173` by default.

## Build

```bash
pnpm build
```

## ngrok

To expose the local dev server via ngrok:

```bash
ngrok http 5173
```
